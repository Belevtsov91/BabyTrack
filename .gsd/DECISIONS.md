## 2026-03-18 — localStorage вместо бэкенда
Проект — персональное PWA приложение для одной семьи.
Нет требований к синхронизации между устройствами, нет серверной части.
localStorage через CRUD-фабрики в crud.ts — достаточно и избавляет от инфраструктурных зависимостей.
Риск: данные пропадают при очистке браузера. Митигация: exportAllData / importAllData в Settings.

## 2026-03-18 — Подход к интеграции redesign
Выбран drop-in replacement подход: файлы из babytrack_redesign_FULL.zip заменяют файлы в src/pages/ один к одному.
Альтернатива (мержить побитно) — слишком трудоёмко для 23 файлов.
Риск: импорты в redesign-файлах могут ссылаться на несуществующие утилиты — нужно проверять перед вставкой.

## 2026-03-18 — Две toast-системы
sonner + @radix-ui/react-toast одновременно.
Причина: shadcn по умолчанию ставит react-toast; sonner добавили позже для более красивых нотификаций.
Оставляем обе — breaking change чтобы убирать.
